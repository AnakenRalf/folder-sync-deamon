# Sync folder deamon

Simple watcher deamon for sync folder.
Splitted to 2 parts:

- watcher
- syncer

## How to use

build watcher
build syncer

For watcher also need config.json file with folder descriprions:

```json
{
  "source_path": "C:\\tmp\\folder_src",
  "network_path": "C:\\tmp\\folder_dst"
}
```

## Watcher

run watcher

## Syncer

run syncer periodically.
